# Note App - Flutter Provider State Management

## Project Overview

## Features

-  **Beautiful UI** - Material Design with colorful notes
-  **CRUD Operations** - Create, Read, Update, Delete notes
-  **State Management** - Provider & ChangeNotifier
-  **Smart Search** - Real-time note searching
-  **Color Picker** - Custom colors for each note
-  **Responsive Design** - Grid layout adapts to screen size
-  **Time Display** - Smart relative time formatting
-  **Safe Delete** - Confirmation dialog before deletion
-  **Sample Data** - Pre-loaded example notes

## Technical Stack

- **Framework**: Flutter 3.19+
- **Language**: Dart 2.19+
- **State Management**: Provider 6.0+
- **Dependencies**:
  - `provider`: ^6.0.5 - State management
  - `intl`: ^0.18.1 - Internationalization
  - `uuid`: ^4.2.1 - Unique ID generation

## Installation & Setup

### Prerequisites
- Flutter SDK 3.19.0 or higher
- Dart SDK 2.19.0 or higher
- Android Studio/VSCode with Flutter extension

### Steps to Run

1. **Clone the repository**
   ```bash
   git clone https://github.com/hanth0509/note_app.git
   cd note_app
   ```

2. **Install dependencies**
   ```bash
   flutter pub get
   ```

3. **Run the application**
   ```bash
   flutter run
   ```
